# IPL Data Analysis using Power BI

## Project Overview

This project analyzes historical **Indian Premier League (IPL)** data using **Microsoft Power BI**.
The objective is to transform raw cricket match data into meaningful insights through **interactive dashboards and visualizations**.

The dashboard helps users understand team performance, player statistics, match trends, and venue analysis using clear visual analytics.

---

## Objectives

* Analyze IPL match and player performance data
* Visualize team statistics and win percentages
* Identify top performing players across seasons
* Analyze venue-based match performance
* Build interactive dashboards using Power BI

---

## Tools and Technologies

* **Power BI Desktop**
* **Power Query**
* **DAX (Data Analysis Expressions)**
* **CSV / Excel Dataset**
* **GitHub**

## Dataset

The dataset contains historical IPL match data including:

* Match ID
* Teams
* Venue
* Season
* Runs scored
* Wickets taken
* Player statistics
* Match results

Source: Kaggle IPL Dataset / Public Cricket Statistics Sources

---

## Data Processing Steps

1. Data Collection from IPL datasets
2. Data Cleaning using Power Query
3. Removing missing values and duplicates
4. Data transformation and formatting
5. Creating calculated columns using DAX
6. Building interactive visual dashboards
